# GitHub Upload Instructions

This folder is the safest version for GitHub's web uploader.

Drag every file in this folder into GitHub. There is no `assets` folder in this version, so GitHub cannot skip nested images.

Upload all of these files together:

- `.nojekyll`
- `index.html`
- `baja-powertrain.html`
- `material-hybrid.html`
- `camping-build.html`
- `sidi-lab.html`
- `combat-robot.html`
- `styles.css`
- `script.js`
- `Yash-Tate-Resume.pdf`
- `hero-yash-tate.jpeg`
- `baja-role-1.png`
- `baja-role-2.jpeg`
- `baja-cad-1.png`
- `baja-cad-2.png`
- `baja-validation-1.png`
- `baja-validation-2.png`

Do not upload the ZIP itself to the website repository unless you are using it only as a download file. For GitHub Pages, GitHub needs the actual HTML, CSS, JS, image, and PDF files.
