# Upload These Files To GitHub Pages

Use this folder for GitHub:

`yash-tate-portfolio-github-upload`

Important: upload the contents of this folder, not the ZIP file itself.

## What To Drag Into GitHub

Drag these files and folders into your repository:

- `index.html`
- `baja-powertrain.html`
- `material-hybrid.html`
- `camping-build.html`
- `sidi-lab.html`
- `combat-robot.html`
- `styles.css`
- `script.js`
- `.nojekyll`
- `assets`

The project pages are at the root level so links like `baja-powertrain.html` work directly on GitHub Pages.

## GitHub Pages Settings

1. Go to your repository.
2. Open **Settings > Pages**.
3. Set source to **Deploy from a branch**.
4. Select `main` and `/root`.
5. Save.

It can take a minute or two for GitHub Pages to update after upload.
