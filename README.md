# Yash Tate Portfolio Website

Static GitHub Pages-ready portfolio for Yash Tate.

## Upload To GitHub Pages

1. Create a new GitHub repository.
2. Upload everything inside this folder, including `index.html`, `styles.css`, `script.js`, and the `assets` folder.
3. In the repository, go to **Settings > Pages**.
4. Set **Source** to `Deploy from a branch`.
5. Select the `main` branch and `/root`, then save.

## Edit Hero Photos

The current hero image is:

`assets/hero-photos/hero-yash-tate.jpeg`

To replace it later, add your new image to `assets/hero-photos` and update the image path in `index.html`.

## Add Project Photos

Put build photos, CAD screenshots, testing images, and motorsport images in:

`assets/gallery`

Then replace the placeholder gallery tiles in `index.html` with `<img>` elements.

## Edit Project Pages

Each selected technical project now has its own page in:

`projects`

Project-specific images and graphs can go in:

`assets/projects`

Each project page has media placeholders under Role, Engineering, Validation, and Impact.

## Add Recommendation Letters

Put letter PDFs in:

`assets/letters`

Then add links in the Recommendation Letters section of `index.html`.
